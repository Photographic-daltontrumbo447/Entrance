# GitHub 镜像发布模板

把 GitLab 的 `v0.3.1-headless-alpha.1` 原样镜像到 GitHub：

- 同一个 tag
- 同一个标题
- 同一份 release notes
- 同一组 zip 和 checksum 附件

> English: Mirror the GitLab `v0.3.1-headless-alpha.1` release to GitHub using the same tag, title, notes, and assets.

## 附件列表

- `entrance-v0.3.1-headless-alpha.1-windows-x64-headless.zip`
- `SHA256SUMS.txt`

## 文案

- 保持 `V0 HEADLESS ALPHA` 表述
- 保持与仓库内文案一致
- 直接使用 `RELEASE_NOTES.md`

> English: Keep the `V0 HEADLESS ALPHA` wording, keep the GitHub wording aligned with the repository docs, and use `RELEASE_NOTES.md` directly.
